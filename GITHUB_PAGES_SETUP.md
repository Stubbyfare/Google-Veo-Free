# GitHub Pages Setup Instructions

## Automatic Setup

1. Go to your repository: https://github.com/Stubbyfare/Google-Veo-Free
2. Click **Settings** → **Pages**
3. Under "Build and deployment":
   - Source: Select **Deploy from a branch**
   - Branch: Select **main** 
   - Folder: Select **/ (root)**
4. Click **Save**

GitHub will automatically deploy your site!

## Your Website URL

After setup completes (1-2 minutes), your site will be available at:

**https://stubbyfare.github.io/Google-Veo-Free/**

## Mobile Access

On mobile, visit: **https://stubbyfare.github.io/Google-Veo-Free/**

You can then:
- Tap the **share icon** (iOS) or **menu** (Android)
- Select **"Add to Home Screen"**
- The app will be installed on your home screen

## What Happens Automatically

Every time you push code to the `main` branch, GitHub Pages will:
- Automatically deploy your changes
- Update the live website in seconds
- Keep your site online 24/7

## Troubleshooting

If the site doesn't appear after 5 minutes:
1. Check that you're on the **main** branch
2. Go back to Settings → Pages
3. Verify the source is set to **Deploy from a branch**
4. Wait another minute and refresh

That's it! Your mobile-friendly video generator is now live!
